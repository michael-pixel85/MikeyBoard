# MikeyBoard

### Entry 1:

Hello. My name is Michael. That brought me to a good idea for the total: Michael -> Mike -> Mikey -> My Key(Board)

I want my first mechanical keyboard to be modeled after this picture. I like the design, the layout, and the simplicity. Although, I originally wanted to add an encoder and a screen on the left side of the keyboard, but since this is my first time, my plan is to make a minimalist keyboard first, then build more advanced designs. 

<img width="480" height="320" alt="image" src="https://github.com/user-attachments/assets/bcfeace2-d88b-4ba2-92d9-9727ed1fb0c6" />

I built my schematic over the course of around two weeks, because I was jumping around days due to a volatile schedule. So, I'm documenting that right now, in my first commit. I did my best to replicate the key design; the only part that might not be too clear are the keys that need stabilizers. Each (normal) key has 2 points between them, but for the ones needing stabilizers (specifically Left Shift, Backspace, and Enter), I tried to leave more space between the keys, around 3 points between. Obviously, the space bar key stands out. 

Honestly, this keyboard design, where the keys are misaligned, does seem a bit intimidating to a first timer like me. I suspect that the PCB design will be a challenge, as I'll have to make precise changes in the alignment of the keys. However, if I figure this out, I'll be able to do many more. It was repetitive, which added time, and the instructions on the Docs were a bit ambiguous/unclear at times, which forced me to go slower. Well, that's it for Entry 1.
